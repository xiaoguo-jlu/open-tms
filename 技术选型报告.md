## 一、技术选型清单

### 1.1 后端技术

| 类别 | 选型 | 说明 |
|------|------|------|
| 框架 | SpringBoot + CXF | RESTful + SOAP支持 |
| 序列化 | Jackson | JSON处理 |
| 数据库 | PostgreSQL | 主数据存储 |
| 容器 | Jetty | 内嵌Web容器 |
| 消息 | RocketMQ | 分布式消息队列 |
| 缓存 | Redis + Redisson | 缓存 + 分布式锁 |

## 二、架构设计

### 2.1 系统架构图

```
┌─────────────────────────────────────────────────────────┐
│                    前端 (Vue3 + Element Plus)            │
├─────────────────────────────────────────────────────────┤
│                      API Gateway                         │
├──────────┬──────────┬──────────┬──────────┬─────────────┤
│ basedata │ cashpool │ dealing  │ workflow │    admin    │
│(基础数据) │ (资金池) │  (交易)  │ (工作流)  │  (后端管理) │
├──────────┴──────────┴──────────┴──────────┴─────────────┤
│                    PostgreSQL + Redis                    │
└─────────────────────────────────────────────────────────┘
```

### 2.2 模块职责

| 模块 | 职责 |
|------|------|
| basedata | 币种/国家/节假日/银行/交易对手/账户管理 |
| cashpool | 现金池配置/自动调拨/内部计息 |
| dealing | 交易录入/审批/查询 |
| workflow | 审批流程/工作流引擎 |
| admin | 认证授权/值集/字典/定时任务/缓存 |
| gateway | 统一入口/鉴权/路由 |

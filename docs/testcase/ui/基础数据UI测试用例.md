| 页面路径 | /basedata/currency |

**前置条件**
1. 前端服务已启动 (localhost:5173)
2. 后端basedata服务已启动 (localhost:8081)

**测试步骤**

4. 验证表格显示币种数据

**预期结果**

- 表格显示币种列表数据
- 分页组件正常显示

**字段规范:**
| 字段 | 类型 | 必填 | 规范 |
|------|------|------|------|
| 业务单元编码 | input | Y | 自动生成或手动输入，唯一校验 |
| 业务单元名称 | input | Y | 最大200字符 |
| 英文名称 | input | N | 最大200字符 |
| 法人代表 | input | N | 最大50字符 |

| 税号 | input | N | 格式校验 |
| 状态 | radio | Y | 默认启用 |

**表单行为:**
- 保存 → 返回列表页
- 保存并新增 → 清空表单继续新增
- 取消 → 确认后返回列表页

## 五、交互规范汇总

### 5.1 通用交互

| 交互 | 规范 |
|------|------|
| 页面跳转 | 平滑过渡，无闪烁 |
| 表单提交 | 加载状态 + 禁用按钮 |
| 批量操作 | 进度条 + 成功/失败统计 |
| 删除确认 | 二次确认对话框 |
| 权限不足 | 提示+不跳转 |

### 5.2 数据校验

| 场景 | 校验规则 |
|------|----------|
| 必填项 | 红色*标记，提交时校验 |
| 格式 | 即时校验，错误提示 |
| 唯一性 | 失焦时查询，冲突提示 |
| 业务规则 | 提交时后端校验 |

### 5.3 响应式

| 断点 | 布局 |
|------|------|
| > 1024px | 完整布局 |
| 768-1024px | 侧边栏收起 |
| < 768px | 单列布局 |

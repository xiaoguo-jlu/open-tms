# Open-TMS M3金融工具模块UX原型

**模块**: M3 金融工具  
**版本**: v1.0  
**日期**: 2026-04-11  
**角色**: UX设计师

---

## 一、模块导航结构

```
M3 金融工具
├── 外汇交易
│   ├── 即期外汇
│   ├── 远期外汇
│   ├── NDF
│   └── 外汇掉期
├── 利率掉期
│   ├── 利率掉期交易
│   └── IRS估值
└── 工具估值
    ├── 在岸产品
    └── 离岸产品
```

## 二、页面清单

| 序号 | 页面 | 路径 | 类型 |
|------|------|------|------|
| 1 | 外汇交易列表 | /fx/deal | 列表页 |

**交易类型:**
| 类型 | 说明 |
|------|------|
| 即期外汇 | Spot |
| 远期外汇 | Forward |
| NDF | 无本金交割远期 |
| 外汇掉期 | FX Swap |

**交易方向:**
| 方向 | 说明 |
|------|------|
| 买入 | Buy |
| 卖出 | Sell |
| 调期 | Swap |

**状态:**
| 状态 | 说明 |
|------|------|
| CREATED | 已创建 |
| EXECUTED | 已执行 |
| SETTLED | 已交割 |
| CLOSED | 已平仓 |

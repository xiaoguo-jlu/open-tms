## 一、模块导航结构

```
M4 风险管理
├── 敞口管理
│   ├── 交易敞口
│   ├── 期限敞口
│   └── 敞口报告
├── 市场风险VaR

├── 套期保值
│   ├── 套期关系
│   └── 套期有效性
└── 减值计算
    ├── 减值模型
    └── 减值准备
```

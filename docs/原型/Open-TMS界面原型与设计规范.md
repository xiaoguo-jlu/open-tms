## 三、导航菜单设计

### 3.1 菜单面板 (Hover 展开)

**触发方式**: 鼠标悬停父级菜单 150ms 后展开

```
┌────────────────────────────────────────────────────────────────────────────┐
│  基础数据                                              [🔍 搜索...]   │
├────────────────────────────────────────────────────────────────────────────┤
│  ┌──────────────────┐ ┌──────────────────┐ ┌──────────────────┐          │
│  │ 👤 业务单元     │ │ 👤 交易员        │ │ 💰 币种          │          │
│  │    管理         │ │    管理          │ │    管理          │          │
│  ├──────────────────┤ ├──────────────────┤ ├──────────────────┤          │
│  │ 🌍 国家/地区     │ │ 📅 节假日        │ │ 🏦 银行          │          │
│  │    管理         │ │   管理           │ │    管理          │          │
│  ├──────────────────┤ ┌──────────────────┐ ┌──────────────────┐          │
│  │ 🤝 交易对手     │ │ 📃 对手方账户   │ │                  │          │
│  │    管理         │ │    管理          │ │                  │          │
│  └──────────────────┘ └──────────────────┘ └──────────────────┘          │
├────────────────────────────────────────────────────────────────────────────┤
│  最近访问: [业务单元管理] [币种管理] [交易录入]                          │
└────────────────────────────────────────────────────────────────────────────┘
```

**面板规格**:
- 宽度: 560px
- 背景: #1E293B (slate-800)
- 定位: 固定在左侧菜单右侧
- 动画: slideInRight 200ms ease-out
- 关闭延迟: 鼠标移出后 200ms 关闭

### 3.2 菜单结构

| 一级菜单 | 二级菜单 (面板内展示) |
|---------|---------------------|
| 数据中心 | 业务单元、交易员、币种、国家/地区、节假日、银行、交易对手、对手方账户 |
| 交易管理 | 交易录入、交易审批、交易查询、交易监控 |
| 资金管理 | 账户余额、资金调拨、资金计划 |
| 风险管理 | 限额管理、风险预警、风险报告 |
| 报表管理 | 业务报表、管理报表 |
| 系统设置 | 用户管理、角色权限、审批流程、系统参数 |

## 七、交互规范

### 7.1 鼠标行为

| 元素 | Normal | Hover | Active |
|------|---------|-------|--------|
| 主按钮 | bg-blue-600 | bg-blue-700 | bg-blue-800 |
| 次按钮 | bg-white | bg-slate-50 | bg-slate-100 |
| 表格行 | bg-white | bg-slate-50 | bg-blue-50 |
| 图标按钮 | text-slate-400 | text-slate-600 | text-blue-600 |

### 7.2 过渡动画

| 场景 | 时长 | 缓动 |
|------|------|------|
| 按钮悬停 | 150ms | ease |
| 菜单展开 | 200ms | ease-out |
| 抽屉滑入 | 200ms | ease-out |
| 弹窗淡入 | 150ms | ease |
| 表格行悬停 | 100ms | ease |

### 7.3 键盘导航

- 所有按钮/输入框: 支持 Tab 聚焦
- 聚焦状态: 2px outline, color: blue-500
- 确认键: Enter 提交表单
- 取消键: Escape 关闭弹窗/抽屉

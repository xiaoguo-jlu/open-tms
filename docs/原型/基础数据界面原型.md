## 四、交互设计

### 4.1 通用交互

| 交互 | 说明 |
|------|------|
| 新增按钮 | 右上角，点击跳转新增页面 |
| 搜索 | 输入条件，点击搜索，支持回车 |
| 重置 | 清空搜索条件，重新查询 |
| 分页 | 支持页码跳转、每页条数选择 |
| 表格排序 | 点击列头排序 |
| 导出 | 导出当前查询结果为Excel |
| 导入 | 下载模板、上传文件、导入校验 |

### 4.2 操作列

| 操作 | 说明 |
|------|------|
| 详情 | 查看完整信息，弹窗或新页面 |
| 编辑 | 跳转编辑页面 |
| 删除 | 二次确认后删除 |

### 4.3 表单校验

| 校验 | 说明 |
|------|------|
| 必填 | 红色*标注，失去焦点校验 |
| 唯一性 | 提交时校验，提示"编码已存在" |
| 格式 | 邮箱/电话等格式校验 |
| 长度 | 最大长度限制 |

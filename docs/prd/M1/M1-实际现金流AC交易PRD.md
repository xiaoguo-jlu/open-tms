## 一、模块概述

**模块名称**: cashflow - 实际现金流管理  
**功能定位**: 管理银行账户的实际现金流(Cashflow)，用于资金头寸监控、对账和流动性管理  
**用户角色**: 资金管理人员、财务人员、资金经理

---

## 二、交易类型定义

| 类型 | 说明 | 优先级 |
|------|------|--------|
| Bank Account Transfer | 由账户转账生成的现金流 | P0 |
| Statement Line | 由银行对账单生成的现金流 | P0 |
| Sweep Transaction | 由资金归集生成的现金流 | P1 |
| Cash Leveling | 由资金平划生成的现金流 | P1 |

## 五、业务规则

1. **编号规则**: 现金流编号 = CF + 账户编号 + 日期 + 序号，如 `CFBJ202604060001`
2. **金额校验**: 金额需大于0，精度匹配币种小数位数
3. **日期校验**: 现金流日期不能晚于当前系统日期
4. **清分规则**:  inflow时对手方账户为资金来源账户，outflow时为资金目标账户
5. **状态流转**: Created → Cleared → Reconciled，不可逆
6. **对账差异**: 系统账与银行账不一致时记录差异，允许手工调账
7. **数据权限**: 用户只能看到有权限的业务单元/银行账户的现金流

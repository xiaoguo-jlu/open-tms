## 一、模块概述

**模块名称**: bankaccount - 银行账户管理  
**功能定位**: 管理企业银行账户信息，支持多银行、多账户类型，支持银企直连获取余额和流水  
**用户角色**: 资金管理人员、IT运维人员

## 三、业务规则

1. **账号唯一性**: 同一银行下账号不可重复
2. **银企配置**: 一个账户只能配置一个银企直连
3. **同步机制**: 按配置频率自动同步余额和流水
4. **对账规则**: 以银行流水为准，系统账与银行账比对
5. **限额控制**: 交易时检查账户日间/日终限额
6. **归集规则**: 归集账户余额自动上划到主账户

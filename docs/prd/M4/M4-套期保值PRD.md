## 一、模块概述

## 三、业务规则

1. **有效性阈值**: 80%≤变动范围≤125%
2. **测试频率**: 至少每季度测试一次
3. **文档要求**: 套保文档需包含目的、策略、有效性评估
4. **会计处理**: 公允价值变动计入其他综合收益

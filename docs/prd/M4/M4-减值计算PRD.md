## 一、模块概述

**模块名称**: impairment - 减值计算

## 三、业务规则

1. **阶段一**: 初始确认后信用风险无显著增加，12个月ECL
2. **阶段二**: 信用风险显著增加，存续期ECL
3. **阶段三**: 已发生信用减值，存续期ECL
4. **迁移触发**: 逾期30天或评级下调2级触发阶段二
5. **前瞻性**: 至少考虑基准/乐观/悲观三情景
6. **PD来源**: 内部评级映射或外部评级映射

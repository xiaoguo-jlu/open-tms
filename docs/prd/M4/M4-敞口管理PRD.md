## 三、业务规则

1. **敞口计算**: 买入为正，卖出为负，净敞口=买入-卖出
2. **限额控制**: 净敞口不得超过限额
3. **计算频率**: 实时计算敞口，每日生成报表
4. **预警阈值**: 敞口超过80%预警，95%告警
5. **数据来源**: 交易系统实时获取交易数据

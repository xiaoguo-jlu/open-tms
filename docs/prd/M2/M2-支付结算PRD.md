## 一、模块概述

**模块名称**: payment - 支付结算  
**功能定位**: 支持多银行支付通道的支付指令处理，包括支付录入、审批、发送和状态追踪

## 三、业务规则

1. **余额校验**: 支付前检查付款账户余额
2. **限额校验**: 支付金额不能超过账户限额
3. **时间校验**: 检查是否在通道服务时间内
4. **重复支付**: 同一指令30分钟内不允许重复提交
5. **金额限制**: 单笔和日累计限额控制
6. **清算时间**: 大额支付T+0，小额支付T+1

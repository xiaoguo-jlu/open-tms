## 一、模块概述

**模块名称**: loan - 贷款交易  
**功能定位**: 支持多种贷款产品的交易录入、审批、执行和还款管理

## 三、业务规则

1. **利率下限**: 贷款利率不低于LPR
2. **担保要求**: 信用贷款需有足够授信额度
3. **抵押率**: 抵押贷款抵押率不超过70%
4. **还款顺序**: 先息后本 / 等额本息 / 等额本金
5. **逾期处理**: 逾期超过90天标记为不良
6. **提前还款**: 提前还款需提前N天申请

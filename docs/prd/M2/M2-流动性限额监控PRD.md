## 一、模块概述

**模块名称**: limitmonitor - 流动性限额监控  
**功能定位**: 对集团流动性风险进行限额监控，支持多维度限额配置和自动预警  
**用户角色**: 资金经理、风险经理

## 三、业务规则

1. **主体限额优先级**: 日间限额优先于日终限额
2. **占用释放时机**: 交易结算完成后释放占用
3. **预警升级**: 黄色预警2小时未处理升级红色
4. **自动调拨**: 仅黑色预警触发自动调拨
5. **限额检查**: 所有支付交易前置检查限额

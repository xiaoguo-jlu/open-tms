## 一、模块概述

**模块名称**: dashboard - 管理驾驶舱  
**功能定位**: 提供资金管理关键指标的可视化展示，支持可配置的仪表盘  
**用户角色**: CFO、资金经理、财务总监

## 三、业务规则

1. **刷新频率**: 默认5分钟刷新，可配置1/5/15/30/60分钟
2. **权限继承**: 继承数据权限，管理员可查看全部
3. **缓存机制**: 减少数据库查询，使用缓存
4. **移动适配**: 支持移动端展示

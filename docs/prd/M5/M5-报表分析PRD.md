**日期**: 2026-04-11

## 一、模块概述

**模块名称**: reporting - 报表分析  
**功能定位**: 提供管理报表、法定报表和自定义报表，支持报表设计和订阅  
**用户角色**: CFO、财务经理、审计人员

---

## 二、功能清单

### 2.1 管理报表

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 资金日报 | 日度资金变动汇总 | P0 |
| 周报/月报 | 周期性分析 | P0 |
| KPI监控 | 资金效率指标追踪 | P0 |
| 排名分析 | 主体/银行排名 | P1 |
| 同比环比 | 同比环比分析 | P1 |

**资金日报字段**:
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 报表日期 | DATE | Y | 报表日期 |
| 期初余额 | DECIMAL(18,2) | Y | 期初余额 |
| 收入 | DECIMAL(18,2) | Y | 现金流入 |
| 支出 | DECIMAL(18,2) | Y | 现金流出 |
| 期末余额 | DECIMAL(18,2) | Y | 期末余额 |
| 净流量 | DECIMAL(18,2) | Y | 净流量 |

## 三、业务规则

1. **数据权限**: 报表数据受数据权限控制
2. **时间要求**: 资金日报T+1生成
3. **导出限制**: 单次导出不超过10000行
4. **订阅时间**: 订阅报表定时发送

---

## 四、验收标准

| 功能 | 验收条件 |
|------|----------|
| 资金日报 | 报表正确生成 |
| 导出功能 | Excel/PDF导出正常 |
| 订阅 | 定时发送成功 |

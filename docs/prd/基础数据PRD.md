## 一、模块概述

**模块名称**: basedata - 基础数据管理

## 三、业务规则

1. **编码唯一性**: 业务单元编码、交易员编号、币种代码、国家代码、银行代码、对手方编号、对手方账户编号均需唯一
2. **级联关系**: 删除国家时需检查关联银行；删除对手方时需检查关联账户
3. **状态控制**: 停用状态的数据在业务功能中不可引用
4. **节假日计算**: 交易交割日需跳过节假日，按节假日表自动调整

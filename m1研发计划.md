## 一、项目概况

### 1.1 MVP功能范围
本次M1版本聚焦**AC交易**和**AT交易**两个简单交易类型：

| 交易类型 | 说明 | 复杂度 |
|----------|------|--------|
| **AC (Actual Cashflow)** | 实际现金流 - 银行流水生成的现金流 | 简单 |
| **AT (Account Transfer)** | 账户转账 - 内部账户间资金划转 | 简单 |

### 1.2 交付要求
| 模块 | 优先级 | 交付标准 |
|------|--------|----------|
| 基础数据 | P0 | 银行账户、币种 |
| AC交易 | P0 | 流水导入 + 清分 + 对账 |
| AT交易 | P0 | 转账创建 + 授权 + 执行 |
| 审批流 | P0 | AT交易二级审批 |

---

## 二、交易类型详情

### 2.1 AC交易 (Actual Cashflow)
```
用途: 银行流水生成的现金流，用于资金池对账
来源: 银企直连 / 手动导入 / AT交易自动生成
状态: Created → Cleared → Reconciled
```

**核心字段**:
- 银行账户、对手方账户
- 流向(Inflow/Outflow)、金额、币种
- 现金流日期、起息日
- 来源类型、来源编号

### 2.2 AT交易 (Account Transfer)
```
用途: 企业银行账户之间的资金划转
类型: 同公司内转账、跨公司转账、同城转账、跨境转账
状态: New → Validated → Authorized → Settled
```

**核心字段**:
- 付款账户、收款账户
- 转账金额、币种
- 预计到账日、支付方式
- 转账类型、是否需要授权

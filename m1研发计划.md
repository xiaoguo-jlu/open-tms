## 二、技术选型

### 2.1 后端技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 框架 | SpringBoot + CXF | RESTful服务 + SOAP支持 |
| 序列化 | Jackson | JSON处理 |
| 数据库 | PostgreSQL | 主数据库 |
| 容器 | Jetty | 内嵌服务器 |
| 消息 | RocketMQ | 消息队列 |
| 缓存 | Redis + Redisson | 分布式锁、缓存 |

### 2.2 前端技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 框架 | Vue3 | 响应式框架 |
| UI组件 | Element Plus | 组件库 |
| 路由 | vue-router | 路由管理 |

### 2.3 模块划分

```
open-tms/
├── basedata      # 基础数据+账户管理（币种/国家/节假日/银行/对手方/账户）
├── cashpool      # 资金池
├── dealing       # 交易
├── workflow      # 工作流
├── admin         # 后端管理（认证授权/值集/字典/定时任务/栏目/缓存）
└── gateway       # API网关
```

| 里程碑 | 周次 | 内容 | 交付物 |
|--------|------|------|--------|
| **M1-1** | 第1周 | 项目启动、环境搭建 | 开发环境、需求评审 |
| **M1-2** | 第2-3周 | 基础数据 + 账户管理 | 功能上线 |
| **M1-3** | 第3-4周 | 资金池 + 自动调拨 | 功能上线 |
| **M1-4** | 第4-5周 | 交易录入 + 审批流 | 功能上线 |
| **M1-5** | 第5-6周 | 测试修复 + UAT | 测试通过 |
| **M1-6** | 第6-7周 | 上线部署 + 验收 | 验收通过 |

## 一、项目概况

### 1.1 MVP功能范围
本次M1版本聚焦4大核心模块：
1. **基础数据** - 币种、国家、节假日、银行、交易对手等
2. **账户管理** - 银行账户维护、余额查询
3. **资金池** - 现金池配置、自动调拨
4. **交易录入** - 存款、贷款、外汇交易及审批

### 1.2 交付要求
| 模块 | 优先级 | 交付标准 |
|------|--------|----------|
| 基础数据 | P0 | CRUD + 前端页面 |
| 账户管理 | P0 | 账户维护 + 余额查询 |
| 资金池 | P0 | 池配置 + 自动调拨 |
| 交易录入 | P0 | 交易创建 + 审批流 |
